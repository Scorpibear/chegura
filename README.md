## Install

- Install LTS version of Node.js
- Download any chess engine, e.g. https://stockfishchess.org/
- From command line:
```
npm install chegura
cd node_modules/chegura
```
- Edit app.config.json and specify path to executable file of chess engine
- From command line:
```
npm start
```
